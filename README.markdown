# Switchback
Fetch urls from Wayback Machine 

## Usage
### With roswell
Fetch all urls from robots.txt files
```
switchback example.com
```

### From lisp
Fetch all urls from robots.txt files
```
(switchback:get-robots "example.com")
```

## Installation
### With roswell
```
$ ros install walpurgisnatch/switchback
```
